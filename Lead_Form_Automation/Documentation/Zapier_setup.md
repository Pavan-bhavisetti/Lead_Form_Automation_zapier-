# Zapier Automation Steps

### 1. Trigger — Google Form
Detects new submission.

### 2. Action — Google Sheets
Stores lead in database.

### 3. Action — Gmail
Sends lead details to your email.

### 4. Action — Telegram
Sends real-time alert to your Telegram using Webhooks.

---

Each step includes:

- Mapping fields
- Testing
- Publishing the Zap
